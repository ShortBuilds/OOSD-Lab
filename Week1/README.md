Overview
This repository contains Java programs designed to refresh and demonstrate Object-Oriented Programming (OOP) concepts. The exercises cover fundamental principles such as Encapsulation, Inheritance, Polymorphism, Abstraction, Method Overloading/Overriding, Constructors, and the use of this and super keywords.

Java is a robust, object-oriented language that organizes software design around objects rather than functions. These exercises illustrate how OOP principles are applied in real-world scenarios.

Lab Exercises
Book Class – Demonstrates encapsulation with validation in setters.

Room & DeluxeRoom – Constructor overloading and inheritance using super.

Hotel Room Booking System – Inheritance and runtime polymorphism.

Abstract Room & Amenities Interface – Abstraction and interface-based design.

Each program is self-contained and highlights a specific OOP concept.
