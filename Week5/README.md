Week 5: Input/Output Streams – Byte Streams & Character Streams
Overview
This repository contains Java programs designed to perform file operations using byte-oriented and character-oriented I/O streams. Byte streams handle raw binary data (images, audio, etc.), while character streams are used for text files.

Topics Covered
Difference between byte and character streams

FileOutputStream

Reading and writing binary files

FileWriter

Lab Exercises
File Copy using Byte Streams – Copies contents of one file to another using FileInputStream and FileOutputStream.

Text File Copy using Character Streams – Reads textual data using FileReader and writes it into another file using FileWriter.
