Java Library – Week 2: Wrapper Classes, Enumeration, and Autoboxing
Overview
This repository contains Java programs designed to explore Java wrapper classes, enumerations, autoboxing, and unboxing. These features allow developers to work seamlessly with primitive data types, object representations, and type-safe constants.

Topics Covered
Wrapper Classes (Integer, Double, Character, etc.)

Autoboxing and Unboxing

Enumeration (enum)

Enum Methods and Constructors

Lab Exercises
Hotel Billing System with Wrapper Classes – Demonstrates autoboxing and unboxing in bill calculation.

Hotel Management System with Enum – Demonstrates enum constructors and methods for room tariff management.
