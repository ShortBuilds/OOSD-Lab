Week 4: Multithreaded Programming – Synchronization
Overview
This repository contains Java programs designed to study thread synchronization and inter-thread communication. Synchronization ensures data consistency when multiple threads access shared resources, while inter-thread communication (wait(), notify()) allows threads to coordinate their execution.
