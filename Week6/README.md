Week 6: Random Access File, Serialization & Deserialization
Overview
This repository contains Java programs designed to demonstrate random access file operations, serialization, and deserialization. These features allow developers to store and retrieve data efficiently, and to persist objects for long-term storage without using a database.

Topics Covered
RandomAccessFile

Serialization

Deserialization

Lab Exercises
Hotel Room Management with RandomAccessFile – Store and access fixed-length room records directly using random access.

Hotel Room Booking with Serialization – Persist room booking objects using serialization and retrieve them with deserialization.
