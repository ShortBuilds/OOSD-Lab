Week 7: Generics in Java
Overview
This repository contains Java programs designed to demonstrate the use of generics in Java. Generics provide type safety and reusability, allowing classes, interfaces, and methods to operate on different data types without explicit casting.

Topics Covered
Generic classes with two type parameters

Bounded types (<T extends Number>)

Generic methods

Generic methods with arrays

Bounded generic methods

Lab Exercises
Generic Class with Two Type Parameters – Store hotel room information with flexible data types.

Generic Method for Room Data – Display room-related data of different types safely.

Bounded Type Parameters for Room Charges – Perform numeric calculations with generics.

Generic Method for Arrays – Manage arrays of room attributes.

Generic Pair Class for Bookings – Associate room numbers with guest details.
