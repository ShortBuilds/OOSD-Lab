Week 3: Multithreaded Programming – Basics
Overview
This repository contains Java programs designed to introduce multithreading and thread lifecycle concepts. Multithreading allows concurrent execution of multiple tasks, improving CPU utilization and responsiveness in applications.

Topics Covered
Thread creation using Thread class

Thread creation using Runnable interface

Thread lifecycle

Thread methods: sleep(), join(), yield()

Lab Exercises
Hotel Room Service Management System – Multiple service requests handled concurrently using threads.

Online Order Processing System – Simulates concurrent order operations with multithreading.
